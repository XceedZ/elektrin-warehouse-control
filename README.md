Repositori ini berisi implementasi User Interface (UI) untuk studi kasus **PT ELEKTRIN**. Proyek ini dibuat untuk menjawab **Pertanyaan No. 6**, yaitu merancang antarmuka untuk:
1.  **Input Barang Retur:** Menangani pengembalian barang dari komisioner (Toko Mitra).

Aplikasi ini dibangun menggunakan **React.js** dan **Tailwind CSS** untuk mendemonstrasikan alur kerja sistem (System Flow) yang dinamis dan responsif.
